### About This Project

This project implements a **Mojaloop Switch Server** — the core interoperability
hub that enables real-time payment transfers between Digital Financial Service
Providers (DFSPs). It is built on top of the Mojaloop open-source platform
(https://mojaloop.io) and follows the FSPIOP API standards.

**Key Components:**
- Mojaloop Hub (v17.2.0)
- Central Ledger Service
- Account Lookup Service (ALS)
- Quoting Service
- Central Settlement Service
- ML API Adapter
- Transaction Requests Service

**Listed in Mojaloop Service Provider Directory:**
https://mojaloop.io/service-provider-directory/entry/1955/
